Blazor

Blazor is a .NET frontend web framework that supports both server-side rendering and client interactivity in a single programming model

  • Blazor fundamentals
    • Overview: 
      Client-side rendering (CSR) means that the final HTML markup is generated by the .NET WebAssembly runtime on the client. 
No HTML for the app's client-generated UI is sent from a server to the client for this type of rendering
Static SSR: The server produces static HTML that doesn't provide for user interactivity or maintaining Razor component state.
Interactive SSR: Blazor events permit user interactivity and Razor component state is maintained by the Blazor framework.
