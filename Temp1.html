Here's a simple and clean way to implement tabs in Blazor (without JavaScript or external libraries).

✅ Result Preview (What You’ll Get)

A UI with clickable tabs, like:

[ Tab 1 ] [ Tab 2 ] [ Tab 3 ]

Content of selected tab

✅ Step-by-Step Implementation

1. Create a Blazor Component for Tabs

SimpleTabs.razor

@code {
    private string activeTab = "Tab1";

    private void SelectTab(string tabName)
    {
        activeTab = tabName;
    }
}

<div class="tabs">
    <button class="@GetTabClass("Tab1")" @onclick="() => SelectTab("Tab1")">Tab 1</button>
    <button class="@GetTabClass("Tab2")" @onclick="() => SelectTab("Tab2")">Tab 2</button>
    <button class="@GetTabClass("Tab3")" @onclick="() => SelectTab("Tab3")">Tab 3</button>
</div>

<div class="tab-content mt-3">
    @if (activeTab == "Tab1")
    {
        <p>This is content of Tab 1</p>
    }
    else if (activeTab == "Tab2")
    {
        <p>This is content of Tab 2</p>
    }
    else if (activeTab == "Tab3")
    {
        <p>This is content of Tab 3</p>
    }
</div>

@code {
    private string GetTabClass(string tabName)
        => activeTab == tabName ? "tab-button active" : "tab-button";
}

2. Add Some CSS Styling

You can place this in wwwroot/css/app.css, site.css, or directly in the SimpleTabs.razor using <style>.

.tab-button {
    padding: 10px 20px;
    margin-right: 5px;
    border: none;
    background-color: lightgray;
    cursor: pointer;
}

.tab-button.active {
    background-color: #007bff;
    color: white;
}

.tab-content {
    padding: 15px;
    border: 1px solid #ccc;
    border-top: none;
}

3. Use It in Your Page

In Pages/Index.razor or any other .razor page:

<h3>Simple Tab Example</h3>
<SimpleTabs />

Make sure to include the @using directive if the component is in a different namespace or folder.

✅ Done!

You now have a fully working tab component in Blazor with clean styling and logic — no JavaScript or external frameworks needed.
